# 🛒 Supermarket Sales Dashboard (Advanced Excel)

## 📌 Project Overview

This project analyzes supermarket sales data to understand sales performance, customer purchasing behavior, and regional distribution of transactions.

The dashboard was developed using **Microsoft Excel**, where raw sales data was cleaned and transformed using **Power Query**, and analyzed using **Pivot Tables and Pivot Charts** to create meaningful business insights through interactive visualizations.

The objective of this project was to transform transactional sales data into an **interactive Excel dashboard** that helps identify trends, top-performing regions, product categories, and customer segments.

---

# 🎯 Business Questions

- What is the **total sales performance** of the supermarket?
- Which **regions generate the highest number of transactions**?
- Which **product categories are sold the most**?
- Which **cities generate the highest sales revenue**?
- What is the **monthly trend of quantity ordered**?
- Which **customer segments purchase the most products**?
- How do **shipping modes impact sales distribution**?

# ⚙️ Project Process

## 1️⃣ Data Collection
- Imported supermarket sales dataset in **CSV format**

## 2️⃣ Data Cleaning (Power Query)
- Removed duplicate records  
- Handled missing values  
- Standardized column names  
- Converted sales and quantity columns to numeric format  
- Extracted **Year and Month fields** for time analysis  

## 3️⃣ Data Analysis (Advanced Excel)
- Created **Pivot Tables** to summarize sales data
- Used **Pivot Charts** for visualizing sales trends
- Built interactive filters using **Slicers**
- Analyzed sales by **region, city, category, and customer segment**

## 4️⃣ Dashboard Development
Designed an interactive Excel dashboard including:

- KPI summary cards
- Region-wise transaction distribution
- Ship mode sales analysis
- Monthly quantity trend chart
- Top 5 cities by total sales
- Category-wise transaction analysis
- Customer segment purchase distribution

---

# 🔍 Key Insights

- 📈 **New York City generated the highest sales revenue** among all cities.
- 🌎 **Central and East regions contribute a major share of total transactions.**
- 📦 **Office Supplies category records the highest number of transactions.**
- 👥 **Corporate customers purchase the highest quantity of products.**
- 📊 Sales activity remains relatively consistent throughout the year with peaks in certain months.
